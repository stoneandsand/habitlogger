* API Endpoints
|---------+---------------------------+------------------------------------+--------------------|
| METHOD  | URL                       | REQUEST                            | RESPONSE           |
|---------+---------------------------+------------------------------------+--------------------|
| GET     | /                         |                                    | html               |
| GET (?) | /login                    |                                    | html               |
| POST    | /login                    | JSON (user, pass)                  | redirect           |
| GET     | /signup                   |                                    | html               |
| POST    | /signup                   | JSON (user, pass, email?)          | redirect           |
| GET     | /username                 |                                    | html               |
| GET     | /api/username/occurrences | JSON (array of objects)            | (array of objects) |
